# Bitcoin nodes scanner

## Deployment 

1.  Clonning

      git clone https://github.com/bitaps-com/btc-nodes-scanner
    
2.  Create postgress container
    
      cd btc-nodes-scanner; cd postgres
      chmod +x run.sh
      ./run.sh
      
3.  Build and run scanner

     cd btc-nodes-scanner
      chmod +x run.sh
      ./run.sh
